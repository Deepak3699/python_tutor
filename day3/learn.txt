✅ What are Operators? (Simple Explanation)

Imagine you’re playing a game.
You use buttons to:

Add score

Compare your level

Check if you have enough coins

Combine powers

In Python, operators are like buttons that tell the computer what to do with values.

Example:
5 + 3 → the + button tells Python to add.



✅ Why do we need Operators?

Because programs need logic like:

Add numbers

Check conditions (>=, ==)

Repeat tasks (with conditions)

Combine logic (and, or)

Change values (+=, -=)

Without operators, coding is impossible.

In Python, .strip() is a string method used to remove unwanted characters (by default, whitespace) from the beginning and end of a string.
How it works:
- text.strip() → removes spaces, tabs, and newline characters from both ends.
- text.lstrip() → removes them only from the left side.
- text.rstrip() → removes them only from the right side.
Examples:
name = "   Alice   "
print(name.strip())   # "Alice"
print(name.lstrip())  # "Alice   "
print(name.rstrip())  # "   Alice"

word = "!!!Hello!!!"
print(word.strip("!"))  # "Hello"
